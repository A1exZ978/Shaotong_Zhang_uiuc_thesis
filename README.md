# Supplementary Materials for Shaotong Zhang's UIUC M.S. Thesis

This repository provides the supplementary CAD models and PLC program resources for Shaotong Zhang's M.S. thesis at the University of Illinois Urbana-Champaign.

## Folder Structure

### 1. 3D model
This folder contains compressed CAD files of the mechanical system developed in the thesis, including the 3D models related to the aircraft door skin handling and vacuum clamping device.

### 2. PLC program
This folder contains compressed PLC project files and related control resources for the control system developed in the thesis.

## Usage Notes

- Please download and extract the compressed files before use.
- For the CAD materials, the project files are provided in STEP (`.stp`) format for convenient access and cross-platform compatibility.
- For the PLC materials, the project files should be opened in the corresponding PLC engineering software environment.
